# Session Handoff

## Workflow

1. Inspect current git status when available.
2. Summarize what changed and why.
3. List files touched.
4. Include commands run and results.
5. Name known issues, assumptions, and unfinished work.
6. State decisions made.
7. Recommend the next task.
8. Include what not to touch when relevant.

## Output

Keep it short enough to paste into a new session:

```text
Project:
Changed:
Files:
Checks:
Known issues:
Decisions:
Next:
Do not touch:
```
